Axon

Axon is a lightweight terminal emulator built using GTK+-2.0 and the VTE terminal widget. It aims to be fast, simple, and light on resources.

Dependencies

Axon requires VTE 2.0 and GTK+ 2.0 or above.

Features

Fast, simple
